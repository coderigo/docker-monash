References and more learning


  1. Docker Machine Create and fire off commands on Docker hosts from your local machine anywhere (cloud, local machine, etc).
  2. Docker Swarm. Native clustering for Docker.
  3. Serf. Cluster self-awareness and event handler. Not part of Docker but useful and interesting to play with.
  4. Weave. Virtual networking of containers running anywhere (multiple hosts, clouds, etc). Very handy.
  5. Docker. THE official site. Full of resources and docs. Great and getting better.
  6. Patience. As with any new tool, there are kinks and things that are hard to get one's head around initially. So a bit of patience and reading glasses is good to have on tap.
